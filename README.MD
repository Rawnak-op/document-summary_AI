# AI Document Summarizer

An AI-powered PDF summarization application built using Streamlit and LangChain. Users can upload technical or research documents and generate summaries with customizable explanation styles and summary lengths.

## Features
- Upload PDF documents
- Extract text automatically
- Multiple explanation styles:
  - Beginner-Friendly
  - Technical
  - Research-Oriented
  - Mathematical
- Adjustable summary lengths
- LLM-powered summarization
- Streamlit-based UI

## Tech Stack
- Python
- Streamlit
- LangChain
- Groq/OpenAI APIs
- PyPDF
